# Mini Blog

Mini Blog is nodescript blog application for my debugging rails by me.

## Features
- Manage users by Google OAUTH
- Authenticate user by Google OAUTH
- Manage blog by author
- Blog can be tagged
- Create sitemap.xml

# System dependencies
- Ruby
- PostgreSQL
- Node
- Google OAUTH

## Optional
- Google Webmaster Tool
- Google Analytics
- Google Tag Manager
